# code-review-spec

用于代码审查、格式化、代码修改。

适用全部语言：Java、Python、Go、TypeScript、Vue 等。

规则：
1. 主规范：`/code-review-spec/spec.md`
2. 细则：`references/*.md`
3. 有 git 看 `git diff`；无 git 全局
4. 先确认范围；强制项有一项不通过即不通过
5. 按规范审查报告格式输出
